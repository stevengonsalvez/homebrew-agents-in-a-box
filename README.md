# homebrew-agents-in-a-box

Homebrew tap for [`ainb`](https://github.com/stevengonsalvez/agents-in-a-box) — the terminal-based dev-environment manager for Claude Code agents.

## Install

```bash
brew tap stevengonsalvez/agents-in-a-box
brew install ainb
```

## Auto-updated

`Formula/ainb.rb` is regenerated on every tagged release of the upstream repo by its release workflow. Do not edit by hand — pushes to `main` here will be overwritten on the next release.
